# 🛒 Shop Management System

A simple and efficient **Shop Management System** built to handle daily business operations such as managing products, purchases, sales, suppliers, and payments through an easy-to-use admin panel.

---

## 🚀 Features

* 📦 **Products Management**

  * Add, update, delete, and view products

* 🗂️ **Categories**

  * Organize products into categories

* 🧾 **Purchases**

  * Record purchases from suppliers
  * Manage purchase items

* 💰 **Sales**

  * Handle customer sales transactions
  * Manage sale items

* 🧑‍💼 **Suppliers**

  * Store supplier information
  * Track supplier relationships

* 💳 **Payments**

  * Supplier payments tracking
  * Sales payments tracking

* 🛠️ **Admin Panel**

  * Simple and clean interface for managing the system

## ⚙️ Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/YoUsEfTaha364/ShopManagement.git
   ```

2. Navigate to the project:

   ```bash
   cd shop-management
   ```

3. Install dependencies:

   ```bash
   composer install
   ```

4. Copy environment file:

   ```bash
   cp .env.example .env
   ```

5. Generate application key:

   ```bash
   php artisan key:generate
   ```

6. Configure your database in `.env`

7. Run migrations:

   ```bash
   php artisan migrate
   ```

8. Start the server:

   ```bash
   php artisan serve
   ```
